# Algoritmos e Estruturas de Dados 

Este repositório foi criado para documentar a minha jornada de aprendizagem na unidade curricular de **Algoritmos e Estruturas de Dados**, com um foco especial em aplicabilidade para **Data Science** e **Negócio**.

## Objetivos
- Implementar conceitos académicos (em C/Java) e traduzi-los para **Python**.
- Analisar a eficiência de cada algoritmo através da notação **Big O**.
- Resolver problemas lógicos complexos aplicados a cenários reais.

## Estrutura do Repositório
* `/teoria`: Notas sobre complexidade, Big O e lógica.
* `/estruturas-lineares`: Implementação de Pilhas, Filas e Listas.
* `/algoritmos-ordenacao`: Métodos de ordenação e busca.
* `/desafios-leetcode`: Resolução de problemas de plataformas LeetCode/HackerRank.

## Tecnologias
- Python 3.x
- Git & GitHub
